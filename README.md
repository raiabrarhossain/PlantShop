# PlantShop (React + Redux)


## Run locally


1. Copy files into a folder named `plantshop`.
2. Run `npm install`.
3. Run `npm start`.


## GitHub
Create a public repo and push this folder. Example commands:


```
git init
git add .
git commit -m "Initial commit - PlantShop"
git branch -M main
git remote add origin <your-github-https-url>
git push -u origin main
```
